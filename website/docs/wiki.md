# Wiki

The project wiki contains community-maintained guides, troubleshooting notes, and extended documentation.

Visit the wiki: [github.com/dayour/copilot_local/wiki](https://github.com/dayour/copilot_local/wiki)

## Contributing to the Wiki

The wiki is enabled on the GitHub repository. Anyone with access can create and edit wiki pages directly through the GitHub UI or by cloning the wiki repo:

```bash
git clone https://github.com/dayour/copilot_local.wiki.git
```
